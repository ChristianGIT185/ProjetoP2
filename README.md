# Sistema de Biblioteca 📚

## Descrição
Este projeto consiste em um sistema de biblioteca desenvolvido em Java com integração ao MySQL. O objetivo é gerenciar o cadastro e a consulta de livros, aplicando conceitos de Programação Orientada a Objetos e banco de dados.

## Funcionalidades
- Cadastro de livros
- Consulta de livros
- Atualização de informações
- Exclusão de registros
- Integração com banco de dados MySQL

## Tecnologias utilizadas
- Java
- MySQL
- JDBC
- Programação Orientada a Objetos (POO)

## Requisitos
- JDK 17 (ou a versão utilizada no projeto)
- MySQL Server
- IDE Java (IntelliJ IDEA, Eclipse ou NetBeans)

## Como executar
1. Clone este repositório.
2. Crie o banco de dados MySQL.
3. Execute o script SQL (caso esteja disponível).
4. Configure as credenciais de acesso ao banco no projeto.
5. Execute a classe `Main.java`.

## Objetivo
Este projeto foi desenvolvido para praticar:
- Programação Orientada a Objetos
- Integração Java com MySQL utilizando JDBC
- Operações CRUD (Create, Read, Update e Delete)
- Organização de código em Java
